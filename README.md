# Smart Attendance Management System

A role-based web application built using **Python Flask** and **MySQL** that allows admins to manage users and view attendance records, while users can securely log in and mark their daily attendance.

## Features
- Secure login with password hashing
- Role-based access (Admin & User)
- Admin can add users and view attendance
- Users can mark and view attendance
- MySQL database integration

## Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS
- Session-based authentication
